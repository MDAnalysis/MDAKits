Installation instructions
=========================

The latest version of maicos can be installed using the following:

.. code-block:: bash

    pip install maicos

The source code of maicos can be installed using the following:

.. code-block:: bash

    pip install git+https://gitlab.com/maicos-devel/maicos.git
