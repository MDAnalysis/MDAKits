Installation instructions
=========================

The latest version of mdachecker can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/IAlibay/CheckStructure@main

The source code of mdachecker can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/IAlibay/CheckStructure@main
