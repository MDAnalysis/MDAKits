Installation instructions
=========================

The latest version of MDANCE can be installed using the following:

.. code-block:: bash

    pip install mdance

The source code of MDANCE can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/mqcomplab/MDANCE@main
