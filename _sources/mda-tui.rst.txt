Installation instructions
=========================

The latest version of mda-tui can be installed using the following:

.. code-block:: bash

    pip install mda-tui

The source code of mda-tui can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/p-j-smith/mda-tui@main
