Installation instructions
=========================

The latest version of propkatraj can be installed using the following:

.. code-block:: bash

    pip install propkatraj

The source code of propkatraj can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/Becksteinlab/propkatraj@main
