| **Description:**
| *Analyze membrane packing defects from MD trajectories using grid-based stamping, leaflet classification, and connected-component clustering. Quantify protein-membrane interactions and contains options for visualization.*



Installation instructions
=========================


The source code of packing_defect can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/jaybraunjr/packing_defect@main
