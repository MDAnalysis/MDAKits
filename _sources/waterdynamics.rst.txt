Installation instructions
=========================

The latest version of waterdynamics can be installed using the following:

.. code-block:: bash

    pip install waterdynamics

The source code of waterdynamics can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/MDAnalysis/waterdynamics@main
