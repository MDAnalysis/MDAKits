Installation instructions
=========================

The latest version of membrane-curvature can be installed using the following:

.. code-block:: bash

    pip install membrane-curvature

The source code of membrane-curvature can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/MDAnalysis/membrane-curvature@main
