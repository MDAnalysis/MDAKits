Installation instructions
=========================

The latest version of qvalue can be installed using the following:

.. code-block:: bash

    pip install md-qvalue

The source code of qvalue can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/cabb99/qvalue@main
