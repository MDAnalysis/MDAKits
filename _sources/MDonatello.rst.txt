Installation instructions
=========================

The latest version of MDonatello can be installed using the following:

.. code-block:: bash

    mamba install -c conda-forge mdonatello

The source code of MDonatello can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/talagayev/MDonatello
