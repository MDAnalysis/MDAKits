Installation instructions
=========================

The latest version of solvation-analysis can be installed using the following:

.. code-block:: bash

    mamba install -c conda-forge solvation_analysis

The source code of solvation-analysis can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/MDAnalysis/solvation-analysis@main
