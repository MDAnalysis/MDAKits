Installation instructions
=========================

The latest version of mdakit_sasa can be installed using the following:

.. code-block:: bash

    pip install mdakit-sasa

The source code of mdakit_sasa can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/pegerto/mdakit_sasa@main
