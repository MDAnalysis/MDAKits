Installation instructions
=========================

The latest version of zarrtraj can be installed using the following:

.. code-block:: bash

    mamba install -c conda-forge zarrtraj

The source code of zarrtraj can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/Becksteinlab/zarrtraj@main
