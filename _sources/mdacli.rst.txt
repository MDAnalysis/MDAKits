Installation instructions
=========================

The latest version of mdacli can be installed using the following:

.. code-block:: bash

    mamba install -c conda-forge mdacli

The source code of mdacli can be installed using the following:

.. code-block:: bash

    pip install git+https://github.com/MDAnalysis/mdacli@main
